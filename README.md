## üöÄ Getting Started

Follow these instructions to get a copy of the project running on your local machine for development and testing purposes.

### Prerequisites

* [Android Studio](https://developer.android.com/studio) (Latest version recommended)
* Java Development Kit (JDK) 11 or higher
* An Android device for testing (casting features may not work fully on emulators)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Ayush1Deshmukh/AllMirrorClone.git](https://github.com/Ayush1Deshmukh/AllMirrorClone.git)
    ```

2.  **Open in Android Studio:**
    * Launch Android Studio.
    * Select **File > Open** and navigate to the cloned directory.

3.  **Sync Project:**
    * Let Gradle build the project and download dependencies.

4.  **Run the App:**
    * Connect your Android device via USB.
    * Click the **Run** ‚ñ∂Ô∏è button (Shift + F10).
